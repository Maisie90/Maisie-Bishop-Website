# Maisie-Bishop-Website
Website showcasing my professional experience and projects
## Live Site

[View the website]([#https://maisie-bishop-website.onrender.com/index.html])

---

## Features

- **Responsive design** that adapts across desktop and mobile devices
- **Modern hero section** with image and gradient glow styling
- **Interactive carousel components** used to display skills and experience
- **Custom styled UI elements** including buttons, navigation, and list styles
- **Reusable layout components** across multiple pages
- **Consistent design system** using CSS variables for colours and styling

---

## Pages

### Home
Introduction, personal summary, and navigation to other sections.

### Professional Experience
A timeline-style layout presenting career history and skills, including interactive carousels to highlight key competencies.

### Projects
Showcases projects built during training and personal development.

---

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Google Fonts (Montserrat)**

---

## Key UI Elements

### Carousel
Custom JavaScript-powered carousels allow users to navigate through experience cards using:

- arrow navigation
- pagination dots
- smooth transitions

### Design System
The site uses a centralised colour system via CSS variables:

- Primary accent
- Secondary accent
- Background layers
- Text hierarchy

This ensures visual consistency across the site.
